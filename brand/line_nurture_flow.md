# LINE OA Nurture Flow — New Rider 3-Message Sequence

The report's Phase 2 deliverable: every new LINE OA follower gets this
sequence. @copy drafts the actual messages from this spec; humans review
and load them into LINE OA broadcast/auto-reply.

## Message 1 — Welcome (immediately on follow)
- Goal: warm welcome + plant the fairness story.
- Content: thanks for adding, one line on what TADA is (เรียกรถ จ่ายแค่
  ค่าบริการ 20 บาท ไม่หักค่าคอมจากคนขับ), one-tap CTA to book first ride.
- Tone: สนุก, friendly, 1–2 emoji. Max 3 lines.

## Message 2 — Education (day 2–3)
- Goal: explain why zero commission matters to the RIDER (happier drivers,
  better service) — not a lecture about our business model.
- Content: one concrete driver story or stat from the brand guide only.
  CTA: try a ride this week.

## Message 3 — Promotion + retargeting (day 5–7)
- Goal: convert. First-ride or weekend incentive (insert current approved
  promo — never invent one), urgency line tied to Bangkok life
  (Friday รถติด, rainy-day surge-free).
- CTA: open app now. Include promo code placeholder {PROMO_CODE}.

## Rules
- Each message ≤3 lines, casual ภาษาพูด, ≥1 emoji, strong CTA.
- All drafts pass human review + LINE Thailand messaging policy before
  sending. Frequency cap: never more than 2 broadcasts/week thereafter.
